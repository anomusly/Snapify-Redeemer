## 🚀 Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/anomusly/Snapify-Redeemer.git
   cd Snapify-Redeemer
   ```
2. **Install Requirements:**

   ```bash
   pip install -r requirements.txt
   ```
3. **Run the tool:**

   ```bash
   python main.py
   ```
